# EagleEye Platform


## Installation
* Install Node.js (https://nodejs.org/dist/v4.4.5/node-v4.4.5-x64.msi)
* Install MongoDB (https://fastdl.mongodb.org/win32/mongodb-win32-x86_64-2008plus-ssl-3.2.7-signed.msi)


## Install application packages:
```sh
npm install
```


## Running application

```sh
start_mongo.bat
start_server.bat
```


## Testing

```sh
unitTest.bat
```
